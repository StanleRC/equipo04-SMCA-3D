# equipo04-SMCA-3D

Deberan usar Temurin 21 y apache tomcat 10.

Cada integrante debera tener su propia rama, ejemplo: rm_stanle (Eviten hacer ramas de cualquiuer otra cosa).

Las modificaciones deberan hacerse en su rama y posteriormente mandarla a dev (la rama main debera estar libre para llevar orden).

NO SUBAN CODIGO NO FUNCIONAL A DEV, solo suban el codigo que ya funcione bien (cualquiera que suba codigo deficiente tendra que arreglarlo por completo)

Esta proibido modificar o eliminar codigo que no sea de ustedes, para eso deberan consultarlo con el autor del codigo (usualmente existe una etiqueta con el nombre de quien realizo esa parte del codigo)

En caso de querer dejar sugerencias en el codigo podran hacerlo mediante un comentario (en javaScript: //Comentario) (en HTML: <!-- comentario --)

Cualquiuer duda consultenlo conmigo o en el grupo de la integradora

suerte jeje :D
